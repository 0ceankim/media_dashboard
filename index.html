📖 이용 가이드

1. 데이터 업데이트
타이틀 옆 업로드 아이콘 클릭 → Google Drive에서 최신 엑셀 다운로드 → 업로드
(애드팝콘 관계자만 Google Drive 접근 가능)

2. 대시보드 전환
좌측 상단 아이콘으로 전체 매체 ↔ 잼팟 & 한게임 리포트 전환

3. 매체별 필터
잼팟 & 한게임 대시보드 하단 표에서 회사/매체/OS 필터 + 정렬 가능

4. 매체 상세 이동
전체 매체 대시보드에서 변동 매체 클릭 → TOP 30으로 이동 (다시 클릭 시 원복)

5. 다크/라이트 모드
우측 상단 아이콘으로 전환

6. 데이터 저장
업로드된 데이터는 브라우저에 개인 저장됨
(다른 사람/시크릿 모드에서는 별도 업로드 필요)

기타 문의 사항 발생시 리워드 사업실 오션에게 연락부탁드립니다!
